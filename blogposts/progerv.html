Programmeer ervaring


Vroeger

Toen ik jonger was werkte ik met Scratch, een programma waarmee je programmeert door blokken op elkaar te stapelen. Iets later zat ik ook kleine spelletjes en programma's te maken met Batch. Ik had vroeger ook een virtual machine gemaakt met MS-DOS (ja serieus) erop en heb daar met Batch Script kleine programma's en malware gemaakt. Later heb ik ook met een Basic Interperter besturingssysteem op de Raspberry Pi kleine programmatjes en spelletjes gemaakt. Ik programmeerde op de raarste manieren ooit toen ik jong was.


Later tot nu

Later ben ik Python gaan leren, en heb daar enorm veel projecten mee gedaan. Nog heel wat later heb ik uit interesse C# geleerd om spellen te maken in Unity. Verder heb ik ook geleerd om met C en C++ te werken, en heb ik ook een klein beetje Assembly geleert uit interesse. Ik had op de middelbare school ook informatica gevolgt. Daar heb ik een beetje HTML en CSS geleerd, en heb ik ook veel SQL geleerd. Ik heb ook Doom mods gemaakt met GZDoom, waarvoor ik ZScript en DECORATE moest leren.
