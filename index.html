Waarom ICT?

Computers zijn gewoon leuk

Ik heb ICT gekozen omdat ik altijd al geïnteresseerd was in computers. Ik heb altijd al vragen gesteld als:

  • Hoe werken besturinssystemen?
  • Hoe start een computer op in de BIOS?
  • Hoe werken programma's en hoe kan het zijn dat code uitgevoerd wordt?
  • Kan ik zelf een besturinssysteem maken?
  • En nog veel meer.....

ik vind met oude computers rondspelen ook leuk. Ik ben zelf ook altijd bezig geweest met hoe ik dingen kan hacken

Ik heb dus ook een rickroll malware in sommige computers op de middelbare school verstopt, van andere computers system32 verwijdert, bestanden van anderen op OneDrive kunnen zien, van sommige computers ook de schijf geformateerd en zelfs op een computer Arch Linux geïnstalleerd met een volledige desktop environment, browser en text editor. Ik werd gelukkig niet geschorst.....

Oh ja en ik heb een besturinssysteem geschreven met een stuiterende DVD logo!!

creativiteit en respect

Ik ben zelf creatief, en zit vol met ideeën binnen en buiten de ICT. Mijn creativiteit kan ook met ICT gecombineerd worden. Ik vind niet alleen mijn ideeën waardevol, maar ook de ideeën van anderen. Zelfs de slechste ideeën zijn waardevol. Ik sta ook altijd klaar om te proberen en te experimenteren. Je weet namelijk nooit wat voor mooie dingen je kan vinden door gewoon te proberen.
